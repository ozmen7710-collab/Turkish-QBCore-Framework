# 🇹🇷 Turkish QBCore Framework
## Kapsamlı QBCore Sunucu Altyapısı

> Professional Turkish-themed FiveM QBCore server framework with complete systems

### 📋 İçerik Listesi

#### 🎮 Core Sistemler
- **HUD & UI System** - Türk temalandırılmış arayüz
- **Job Systems** - İş scriptleri (Taksi, Polis, Doktor, vb.)
- **Vehicle System** - Araç yönetimi ve showroom
- **Property System** - İşletme ve emlak sistemi
- **Clothing System** - Türk tarzı kıyafet sistemi
- **NPC System** - Türk karakterleri ve dialog sistemi
- **Location & Map** - Türk şehirleri teması
- **Audio System** - Türk müzikleri ve ses efektleri
- **Database Config** - Tam MySQL entegrasyonu
- **Language Pack** - Türkçe dil dosyası

### 🚀 Kurulum
1. Repository'yi klonlayın
2. `sql` klasöründeki veritabanı dosyalarını import edin
3. `resources` içindeki tüm klasörleri `resources` dizinine kopyalayın
4. `server.cfg` dosyasını güncelleyin
5. Sunucuyu başlatın
